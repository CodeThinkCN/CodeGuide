# CodeThink 代码规范

如果你希望编写 `CodeThink` 的代码，你的代码应该满足本仓库文件所描述的规范。否则 `Pull Request` 类操作可能被拒绝。

组织代码规范是衡量一个组织专业程度的重要标准之一，也有利于形成统一的编码习惯，利于阅读和理解代码。
